# 개발 관련


## 환경설정 파일 `./.env`

```
# ./.env
DISCORD_BOT_TOKEN=your-token-here
DISCORD_CLIENT_ID=your-bot-client-id
SQLITE3_FILE=database.sqlite
...
```

## 실행 순서

1. `/` 명령어 등록

```bash
yarn cmd-dev
```

2. 봇 실행
```bash
yarn dev
```
